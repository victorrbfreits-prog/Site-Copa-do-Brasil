# Site-Copa-do-Brasil

Projeto desenvolvido como atividade acadêmica do curso de Informática do IFRN.

## Sobre o projeto

Este site foi criado com o objetivo de praticar os fundamentos do HTML e a estruturação de páginas web utilizando elementos semânticos. O conteúdo é baseado na Copa do Brasil e apresenta informações organizadas por meio de títulos, parágrafos, listas, imagens e links.

## Conceitos praticados

- Estrutura básica do HTML5
- Elementos semânticos (`header`, `main` e `footer`)
- Hierarquia de títulos (`h1` e `h2`)
- Parágrafos e organização de conteúdo
- Listas ordenadas e não ordenadas
- Inserção de imagens
- Criação de links de navegação

## Tecnologias utilizadas

- HTML5

## Objetivo

Desenvolver habilidades em desenvolvimento FULL STARK e registrar minha evolução como estudante de Informática no IFRN.
